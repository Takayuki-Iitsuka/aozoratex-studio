# 縦書きPDF_視認性・配色設定

スマートフォンやPCでの長文読解、特に青空文庫のような縦書きテキスト（明朝体）を読む際に、網膜への刺激を和らげ、集中力を持続させるための配色（Color Scheme）と視認性（Visibility）の基準をまとめます。

## コントラスト比（Contrast Ratio）の基準
W3CのWCAG (Web Content Accessibility Guidelines) に基づき、背景（Background）と文字（Foreground）のコントラスト比を計算します。
- 読解に必要な最低基準: **4.5:1** (Level AA)
- 理想的な範囲: **7:1 〜 10:1**

**注意（ハレーションのリスク）:**
コントラスト比が最大の 21:1 （完全な黒 `#000000` 背景に完全な白 `#FFFFFF` 文字）の組み合わせは、有機EL（OLED）ディスプレイ等において明朝体の細い横線がバックライトで飛んでしまう「ハレーション現象」を引き起こすため、長時間の読書には推奨されません。

## 配色バリエーション（厳選カテゴリ）

生成された多数の配色の中から、実用性とテーマ性を兼ね備えた代表的な組み合わせカテゴリを定義します。

### 1. 和色（日本の伝統色）
青空文庫や近代文学の読解に最適化された、和紙や伝統的な布を模した配色です。
- **白練 (Shironeri) x 墨 (Sumi):** `BG: #F5F5F0` | `FG: #2B2B2B`
- **生成 (Kinari) x 紫黒 (Shikoku):** `BG: #EAE4D4` | `FG: #261E1C`

### 2. 漢色（古代中国の色彩）
『史記』などの漢文や史料を読むために、竹簡や帛書（絹）などの歴史的な記録媒体の質感を模倣します。
- **竹簡 (Bamboo Slip) x 甲骨 (Oracle Bone):** `BG: #E8D3A2` | `FG: #4A3B2C`
- **帛書 (Silk Text) x 朱砂 (Cinnabar):** `BG: #F5E8C7` | `FG: #2B1D1D`

### 3. 将棋と盤上工芸
榧（かや）や黄楊（つげ）といった伝統的な木材と、漆（うるし）のコントラストを応用した、高い集中力を要する読解向け配色です。高コントラストをわずかに抑えることで眼精疲労を防ぎます。
- **榧の盤 (Kaya) x 漆の文字 (Lacquer):** `BG: #E0B56A` | `FG: #0D0B0A`
- **黄楊の駒 (Buxus) x 彫埋駒の陰 (Hori-ume):** `BG: #F0D697` | `FG: #2C2519`

### 4. 自然と鉱物
四季の風景や天然石の光の反射率を模倣し、デジタルデバイス特有のブルーライトを心理的・物理的に緩和します。
- **翡翠 (Jade) x 瑠璃 (Lapis):** `BG: #2A4B46` | `FG: #1B263B` （Dark Mode 反転例）

### 5. UI / 端末環境
電子書籍リーダー（e-ink）や、モダンな開発環境（VS Code のテーマ等）で長年テストされてきた実用的な配色です。
- **Kindle Sepia:** `BG: #F4ECD8` | `FG: #5B4636`
- **Gruvbox Material (Dark):** `BG: #32302F` | `FG: #D4BE98`

---

## 実装に関するメモ
- 実際のアプリケーション実装においては、これらの `Background` と `Foreground` のHEX値（#RRGGBB）を `settings.ini` または HTML の UI からユーザーが選択し、LaTeX コンパイル時に変数値として動的に注入（Inject）する仕組みを構築します。
- これまでに生成した200パターンの配色は、統合HTML（`colors.html`）の実装としてフロントエンド側に委ねることとします。

## 現在の出力カラー・テーマの種類（コード実装）

色の指定には **3つの層** があります。

### 🎨 第1層：`--mode`（基本モード）— **4 種類**

コマンドラインや WebUI から指定する大枚のモードです。

| # | モード名 | 概要 |
|---|---|---|
| 1 | `light` | 明るい背景 × 暗い文字 |
| 2 | `dark` | 暗い背景 × 明るい文字 |
| 3 | `intermediate` | `settings.ini` の `background_color_intermediate` / `text_color_intermediate` 値を使用（デフォルト：薄灰 `#D3D3D3` × 濃灰 `#4F4F4F`） |
| 4 | `sepia` | `color-palettes.json` の `preset_modes` を使用。Kindle Sepia・古紙色の 2 パターン（**WebUI専用**） |

> [!NOTE]
> `sepia` は `aozoratex.py` の `--mode` 引数の `choices` には列挙されていませんが、`app.py` の `_build_color_schemes()` でプリセットとして処理されます。**WebUI 専用のモード**です。

### 🎨 第2層：`color-palettes.json` のカテゴリ— **5 カテゴリ**

`light` / `dark` モード選択時に、コントラスト比 ≥ 4.5 を満たす組み合わせが自動生成されます。

| カテゴリ | Light パレット（背景候補） | Dark パレット（前景候補） |
|---|---|---|
| **和色** | 白練・桜色・白緑・鳥の子・生成・椉色（6色） | 墅・呂色・鉄御納戸・海松色・黒櫖・紫黒（6色） |
| **漢色** | 玉色・竹簡・帛書・象牙・浅黄・胡麻（6色） | 甲骨・朱砂・瑧璃・青銅・翡翠・黒茶（6色） |
| **将棋** | 榄の盤・黄杨の駒・桂の盤・駒台・和紙の棋譜（5色） | 漆の文字・彫埋駒の陰・盛上駒の良・布盤・桑の駒笥（5色） |
| **自然** | 春の空・薄荷・朝露・象牙・麦穂・砂浜（6色） | スレート岩・オリーブ・サドルレザー・真夜中・海の底・大樹（6色） |
| **UI/端末** | Pure White・Gray 50/100・Light Blue・OLED Light（5色） | Gray 700/800/900・Material Dark・AMOLED Black（5色） |

### 🎨 第3層：`--bg-color` / `--fg-color`（完全自由指定）

`#RRGGBB` 形式で **任意の色を直接指定** できます。パレット外の色も使用可能です。

### 📝 まとめ

| 選択方法 | 種類 |
|---|---|
| 基本モード (`--mode`) | `light` / `dark` / `intermediate` / `sepia`（WebUI限定） |
| パレットカテゴリ | 5 カテゴリ（和色・漢色・将棋・自然・UI/端末） |
| 自由指定 (`--bg-color` / `--fg-color`) | 無限制（任意の `#RRGGBB`） |
