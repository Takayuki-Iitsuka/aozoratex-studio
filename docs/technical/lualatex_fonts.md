## settings.ini コピペ用フォントリスト

`settings.ini` の `font =` の行にそのままコピー＆ペーストできるフォント名の一覧です。  
行末の `;` 以降はコメントです（`settings.ini` のインラインコメント記号）。

```ini
; ==================================================
; settings.ini コピペ用フォントリスト
; 使い方: 各デバイスセクション [PDF_*] の font = 行を
;         下記から選んでコピー＆ペーストしてください。
; ==================================================

; ---- 明朝体（縦書き・本文向け・推奨順） ----
font = Yu Mincho              ; 游明朝（Windows 10/11 標準・細線が美しい）
; font = 游明朝               ; 游明朝（日本語名での指定が必要な場合）
; font = 源ノ明朝              ; Source Han Serif JP（Adobe・OFL）
; font = 源ノ明朝 JP           ; Source Han Serif JP（地域バリアント）
; font = Noto Serif JP         ; Noto Serif JP（Google・OFL）
; font = BIZ UDPMincho         ; BIZ UDP明朝（Microsoft・Windows 11+）
; font = BIZ UDP明朝           ; BIZ UDP明朝（日本語名）
; font = BIZ UD明朝            ; BIZ UD明朝（等幅版）
; font = IPAex明朝             ; IPAex明朝（IPA標準・OFL）
; font = IPA明朝               ; IPA明朝（等幅版）
; font = IPA P明朝             ; IPA P明朝（プロポーショナル版）
; font = Shippori Mincho       ; しっぽり明朝（OFL・本文向け）
; font = Shippori Mincho B1    ; しっぽり明朝 B1（OFL）
; font = Zen Old Mincho        ; 禅旧字体明朝（OFL）
; font = Hina Mincho           ; 雛明朝（OFL・細めで優雅）
; font = ほのか新明朝          ; ほのか新明朝（Light）
; font = コーポレート明朝 ver2  ; コーポレート明朝（Medium）
; font = さつき源代明朝        ; さつき源代明朝（Medium）
; font = 源暎こぶり明朝 v6     ; 源暎こぶり明朝（OFL）
; font = 源暎ちくご明朝 v3     ; 源暎ちくご明朝（OFL）
; font = ＭＳ 明朝             ; MS明朝（Windows標準・等幅）
; font = ＭＳ Ｐ明朝           ; MS P明朝（プロポーショナル版）

; ---- ゴシック体（見出し・注釈・サンセリフ向け） ----

; font = IPAゴシック           ; IPAゴシック（等幅版）
; font = 源ノ角ゴシック Code JP ; Source Han Code JP（プログラミング向き）
```

> [!NOTE]
> **使用例（settings.ini）：**
> ```ini
> [PDF_iphone]
> font = 源ノ明朝
> font_size = 11pt
> characters_per_line = 33
> ```
>
> フォント名にスペースや日本語が含まれる場合もそのまま記載できます（`luatexja-fontspec` が対応）。

> [!TIP]
> **フォントが見つからない場合の確認方法：**
> ```powershell
> # TeX Live の fc-list でシステムフォントを確認
> fc-list :lang=ja family | sort
> ```
> 出力されたフォント名をそのまま `font =` に指定してください。
