\newlength{\GeoRuleWidth}
\newcommand{\GeoRule}[1]{%
  \setlength{\GeoRuleWidth}{#1}%
  \begin{tikzpicture}[baseline=-0.6ex, x=1pt, y=1pt]
    \pgfmathsetmacro{\TW}{\strip@pt\GeoRuleWidth}
    % 左端細線
    \draw[line width=0.3pt, color=rulegray](0,0) -- (0.17*\TW, 0);
    % 左側 小矩形×3
    \foreach \i in {0,1,2}{
      \draw[line width=0.45pt, color=rulegray, fill=rulegray!18]
        ({0.17*\TW + \i*9.5}, -1.6) rectangle ++(5, 3.2);
    }
    % 中央 菱形
    \filldraw[color=rulegray, fill=rulegray!55]
      ({0.5*\TW},  3.2) -- ({0.5*\TW+3.2}, 0) --
      ({0.5*\TW}, -3.2) -- ({0.5*\TW-3.2}, 0) -- cycle;
    % 右側 小矩形×3
    \foreach \i in {0,1,2}{
      \draw[line width=0.45pt, color=rulegray, fill=rulegray!18]
        ({0.83*\TW-(2-\i)*9.5}, -1.6) rectangle ++(5, 3.2);
    }
    % 右端細線
    \draw[line width=0.3pt, color=rulegray](0.83*\TW, 0) -- (\TW, 0);
  \end{tikzpicture}%
}

\chapter*{走れメロス}
\thispagestyle{plain}

メロスは激怒した。必ず、かの邪智暴虐の王を除かなければならぬと決意した。メロスには政治がわからぬ。メロスは、村の牧人である。笛を吹き、羊と遊んで暮して来た。けれども邪悪に対しては、人一倍に敏感であった。

きょう未明メロスは村を出発し、野を越え山越え、十里はなれた此のシラクスの市にやって来た。メロスには父も、母も無い。女房も無い。十六の、内気な妹と二人暮しだ。この妹は、村の或る律気な一牧人を、近々、花婿として迎える事になっていた。結婚式も間近かなのである。メロスは、それゆえ、花嫁の衣裳やら祝宴の御馳走やらを買いに、はるばる市にやって来たのだ。先ず、その品々を買い集め、それから都の大路をぶらぶら歩いた。メロスには竹馬の友があった。セリヌンティウスである。今は此のシラクスの市で、石工をしている。その友を、これから訪ねてみるつもりなのだ。久しく逢わなかったのだから、訪ねて行くのが楽しみである。

ところが、まちの様子が、おかしい。ひっそりしている。もう既に日も落ちて、まちの人たちは、めいめい家に引きとって、静かに戸を閉めていた。たまに通る人も、首をすくめ、そそくさと急いでいる様子で、それは何か不吉なものを予感させた。

メロスは、それでも知らん振りして、まちの大路を歩いた。するとそこに一人の老爺が、うずくまっていた。

「ご老体、なぜ泣いているのですか。」

老爺は、しかし、返事もせず、ひたすら地面を見つめていた。

「何かあったのですか？」

老爺は、やっと頭を上げ、ぼそぼそと告げた。「今から三日前のことじゃ。王様が、またひとり家来を殺されました。」

「なぜ？」

「邪推からです。人の心が信じられぬのです。」

メロスは拳を握りしめ、王城へと歩き出した。
