配色設計ガイド

以下の資料を統合: 青空文庫PDF化配色提案 / 視認性の高い配色一覧と考察 / スマートフォン読書用配色バリエーション拡張 / 200配色生成メモ群。

設計方針

  • 背景色と文字色のコントラストは、可読性と眩しさのバランスを優先。
  • ライト / ダーク / セピアを中心に、色パレットJSONを単一ソースとして管理。
  • UI、CLI、配色一覧ページで同じJSONを参照して整合を保つ。

代表モード

mode用途備考
light日中読書明るい背景 + 暗い文字
dark夜間読書暗い背景 + 明るい文字
sepia長時間読書暖色系で刺激を抑える
