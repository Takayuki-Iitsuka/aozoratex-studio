変換・文字コード・LaTeX設定ガイド

以下の資料を統合: LualaTexについて / Tex化について / 文字コード / ファイルのエンコード / ページ番号設定 / 高度な読書体験。

基本フロー

  1. HTML/XHTML を読み込み（エンコーディング判定 + フォールバック）
  2. 本文ノードを LaTeX 断片へ変換（ルビ・見出し・画像など）
  3. テンプレートに埋め込み、TeX出力
  4. 必要時に lualatex でPDF化

文字コード運用

  • 入力は UTF-8 / Shift_JIS / CP932 が混在し得るため、自動判定に加えて --encoding で明示可能にする。
  • 内部処理は UTF-8 に正規化して扱う。

CLI 例

.venv\Scripts\python.exe -m src.aozoratex data/1567_14913.html --device iphone --mode light
.venv\Scripts\python.exe -m src.aozoratex data --device ipad_landscape --out out/ipad_landscape

PDF化（Web API）

/api/generate に compile_pdf=true を渡すと、TeX生成後に lualatex を実行して PDF を生成します。
