デバイス別組版ガイド

本ページは、デバイスごとのページサイズ・行数・組版モードを統一基準で運用するための詳細ガイドです。設定値の調整時は、まず本ページの標準プロファイルを基準にしてください。

1. 標準プロファイル

2. 運用ルール

  • 設定ソースは config/device_settings.default.ini を基準にし、環境差分は config/device_settings.custom.ini で上書きする。
  • ipad_landscape は two_column を前提とし、本文開始時に \twocolumn、奥付前で \onecolumn を適用する。
  • 余白や文字数の変更は、まず 1 デバイスで効果確認後に他デバイスへ展開する。

3. 調整手順（推奨）

  1. 対象デバイスを決め、現行値を確認する。
  2. device_settings.custom.ini に最小差分で上書きする。
  3. Web UI でプレビューし、改ページ・可読性・装飾表示を確認する。
  4. 必要なら CLI でも同条件生成して再現性を確認する。
