配色・テーマの種類

色は3つの階層で選べます。順に説明します。

第1層：基本モード（--mode）

最初に選ぶ大きな分類です。4種類あります。

モード 見た目 おすすめの使い方
light 白背景に黒文字（標準） 一般的な印刷・画面表示
dark 黒背景に白文字 夜間閲覧や暗い環境向け
intermediate 薄い灰色背景 × 濃い灰色文字 中間的な柔らかい印象
sepia 古紙のようなセピア色 読み物らしい温かみを出したいとき（主にWeb画面）

※ sepia は主に画面（Web UI）で使われます。

第2層：カラーパレット（color-palettes.json）

基本モードを選んだあと、さらに「雰囲気」を決めるための色グループです。以下の5カテゴリがあります。

カテゴリ 明るい背景の例 暗い文字の例（Dark時）
和色 白練、桜色、鳥の子 など 墨、呂色 など
漢色 玉色、竹簡、象牙 など 甲骨、瑠璃 など
将棋 榧の盤、黄楊の駒、和紙 など 漆の文字、布盤 など
自然 春の空、麦穂、砂浜 など オリーブ、真夜中 など
UI/端末 Pure White、Light Blue など Material Dark、AMOLED Black など

第3層：自由に色を指定（--bg-color / --fg-color）

上記以外にも、直接 #RRGGBB の形式で好きな色を指定できます。
例：--bg-color #f5f5dc --fg-color #2f2f2f

選び方のまとめ

選び方 内容
基本モード light / dark / intermediate / sepia の4つから選ぶ
パレットカテゴリ 和色・漢色・将棋・自然・UI/端末の5グループ
完全自由 任意のHEXカラー（#RRGGBB）
