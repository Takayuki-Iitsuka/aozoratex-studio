現在の出力サイズ（デバイス）の種類

現行バージョンでは、config/device_settings.default.ini に定義された 3 種類の標準デバイスを使用します。

No. device 基準サイズ(mm) 用途
1 smart 62.0 x 134.0 iPhone 11 Pro 基準のスマートフォン表示
2 tablet 158.0 x 227.0 iPad 基準。Portrait / Landscape を切替可能
3 pc 210.0 x 297.0 A4 出力

向き（Orientation）

device 対応向き 説明
smart portrait / landscape 基準は portrait。landscape 指定時は幅・高さを入れ替えて出力。
tablet portrait / landscape Web UI で向き選択が可能。landscape 時は 2 段組を既定値とする。
pc portrait A4 固定。

補足

  • CLI の --device 既定値は smart です。
  • Web UI / CLI / Make は同じデバイス設定（config/*.ini）を参照します。
  • 旧デバイス名（iphone / android / ipad / ipad_landscape）は内部で互換変換されます。

端末別の生成先

CLI/Web/Make いずれも、端末別の生成先は以下で統一されます。

device TEX と中間生成物 PDF
smart out/work/smart/ out/pdf/smart/
tablet out/work/tablet/ out/pdf/tablet/
pc out/work/pc/ out/pdf/pc/

出力ファイル名は 元ファイル名_タイトル_作者名 形式です。
