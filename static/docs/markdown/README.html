Docs Index

本ページは、docs/ ディレクトリの構成方針を示すインデックスです。運用情報（manuals）と技術情報（technical）を分離し、参照目的に応じて検索しやすい形で整理します。

manuals

利用者向け・運用者向けの手順書を格納します。日常運用、設定変更、生成操作など、再現可能な手順を中心に記述します。

technical

実装仕様、設計判断、検討メモ、調査結果を格納します。コード変更時の根拠資料として参照できる粒度で記述します。

運用ルール

  • 運用判断が必要な項目は manuals を優先する。
  • 仕様背景や実装理由の確認は technical を参照する。
  • 最新の実行手順はリポジトリルートの README.md を基準とする。
  • ブラウザ版ドキュメントは /docs（static/docs/）から参照する。
