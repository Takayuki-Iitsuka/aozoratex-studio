フォント選定ガイド

史学・古典系の文書では、Readability（可読性） に加えて Compatibility（外字・異体字への対応力） が重要です。 本ページは、本文と見出しの運用方針を一貫した基準でまとめた公式ガイドです。

選定方針

学術用途では、出現頻度が低い文字でも正しく描画できることを最優先にします。 そのうえで、Aesthetics（審美性） を評価し、文書全体の読みやすさと雰囲気を調整します。

評価軸 内容 運用上の扱い
Readability（可読性） 長文読解時の疲労の少なさ、字面の安定性 本文フォント選定の基本軸
Compatibility（互換性） 外字・異体字・拡張漢字の収録範囲 学術文書では最優先
Aesthetics（審美性） 文学的な雰囲気、見出しの印象形成 互換性を満たした上で調整

既定（デフォルト）推奨

既定フォントは IPAmjMincho を推奨します。 Compatibility（互換性） が高く、文字欠けリスクを最小化できるためです。

推奨運用（ハイブリッド方式）

学術的な正確性と見た目を両立するには、本文フォントと補完フォントを使い分ける Hybrid markup（ハイブリッド運用） が有効です。

  • 通常本文は Shippori Mincho か Yu Mincho を使い、読み味を優先する。
  • 外字・異体字を含む箇所のみ IPAmjMincho で局所的に補完する。
  • 最終出力前に、文字欠けとフォント混在箇所を目視で確認する。
% 例: 本文を Shippori Mincho、外字のみ IPAmjMincho で補完
\setmainjfont{Shippori Mincho}
\newfontfamily\academicfont{IPAmjMincho}
本文中で特殊字が必要な箇所は {\academicfont 𠮷} のように局所指定する。

コピペ方法

「コピペ用コマンド」列のテキストをプリアンブルに貼り付けるだけで使用可能。
和文フォントには \setmainjfont、欧文には \setmainfont を使用。

TTC について : TTC（TrueType Collection）は複数フォントを含むコンテナ。
LuaLaTeX では [FontIndex=0] 等でスタイルを選択可能。

フォント変更の手順（まとめ）

  1. 上記の表でフォントを選ぶ
  2. 「コピペ用コマンド」列のテキストをそのままコピー
  3. 自分の .tex ファイルのプリアンブルに貼り付ける
% ▶ 和文フォント（明朝・ゴシック）
\setmainjfont{フォント名}  % 本文明朝体
\setsansjfont{フォント名}  % ゴシック体（見出し等）

% ▶ 欧文フォント（セリフ・サンセリフ・等幅）
\setmainfont{フォント名}   % 本文欧文
\setsansfont{フォント名}   % サンセリフ欧文
\setmonofont{フォント名}   % 等幅（コード等）

% ▶ ウェイト（太さ）指定例
\setmainjfont{Harano Aji Mincho}[
  BoldFont     = {Harano Aji Mincho},
  BoldFeatures = {FakeBold=2.5}  % 合成太字（Bold がない場合）
]
