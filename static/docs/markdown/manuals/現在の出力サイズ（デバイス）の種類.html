現在の出力サイズ（デバイス）の種類

現行バージョンでは、config/device_settings.default.ini に定義された 5 種類の標準デバイスを使用します。

No. device サイズ(mm) 用途
1 iphone 65.0 x 140.0 スマートフォン縦書きの既定プロファイル
2 android 70.0 x 155.5 Android 現代型
3 ipad 158.0 x 227.0 iPad 縦向き（単段組）
4 ipad_landscape 227.0 x 158.0 iPad 横向き（二段組）
5 pc 210.0 x 297.0 A4 出力

補足

  • CLI の --device 既定値は iphone です。
  • Web UI も同じ標準デバイス群を参照します。
  • カスタムデバイスを追加する場合は config/device_settings.custom.ini に [<device>] セクションを追加します。

端末別の生成先

CLI/Web/Make いずれも、端末別の生成先は以下で統一されます。

device TEX と中間生成物 PDF
iphone out/work/iphone/ out/pdf/iphone/
android out/work/android/ out/pdf/android/
ipad out/work/ipad/ out/pdf/ipad/
ipad_landscape out/work/ipad_landscape/ out/pdf/ipad_landscape/
pc out/work/pc/ out/pdf/pc/

出力ファイル名は 元ファイル名_タイトル_作者名 形式です。
