デバイス別_縦書きPDF最適化設定

AozoraTeX Studio では、デバイスごとにページサイズ・余白・1行文字数を切り替えて縦書き可読性を最適化します。

1. 標準デバイスプロファイル

device width x height (mm) 文字数目安 備考
iphone 65.0 x 140.0 33 既定デバイス
android 70.0 x 155.5 30 やや広めのスマートフォン画面
ipad 158.0 x 227.0 35 単段組
ipad_landscape 227.0 x 158.0 35 二段組（two_column）
pc 210.0 x 297.0 40 A4

2. 組版ルール

  • 行送りと字間は JIS/JLREQ 準拠値をコード側で適用します。
  • 色設定は light/dark/intermediate を切り替えます。
  • 本文 Frame（フレーム・枠）は pc / ipad / ipad_landscape のみ有効化できます。

3. カスタマイズ手順

  1. config/device_settings.custom.ini に対象デバイスセクションを作成
  2. font_size、characters_per_line、余白（margin_*_mm）を調整
  3. Web UI で表示確認し、必要に応じて配色と装飾も合わせて調整

4. 例: iPad 横向きで二段組最適化

[ipad_landscape]
font_size = 13.5
characters_per_line = 35
width_mm = 227.0
height_mm = 158.0
mode = two_column
show_page_number = true
