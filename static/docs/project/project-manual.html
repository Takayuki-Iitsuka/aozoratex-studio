プロジェクト概要・運用

本ページは以下の資料を統合した運用ガイドです。

  • static/docs/markdown/manuals/*.html
  • static/docs/markdown/technical/Todo.html
  • static/docs/markdown/technical/*.html（設計資料）

目的

青空文庫HTMLを LuaLaTeX で PDF 化し、デバイス別の読みやすさを確保すること。Web UI から配色とデバイスを選択し、TeX/PDF 生成まで実行できます。

現在の構成方針

  • 起動エントリ: python -m src.aozoratex, python -m src.aozora_server
  • 設定: config/global_settings.*.ini / config/device_settings.*.ini
  • 実装本体: src/
  • 入力: data/
  • 出力: out/
  • Web静的資産: static/
  • Web UI エントリ: static/index.html
  • LaTeXテンプレート: src/aozoratex.py 内の埋め込みテンプレート定数で管理
  • ツール群: tools/
  • 変換済みドキュメント: static/docs/markdown/

推奨運用

  • 通常利用は Web UI（python -m src.aozora_server）か CLI（python -m src.aozoratex）を使う。
  • 生成物の中間ログや一時ファイルは out/ 以下へ集約する。
  • 仕様メモは static/docs/markdown 配下の該当ページを更新する。
