プロジェクト概要・運用

本ページは以下の資料を統合した運用ガイドです。

  • docs/manuals/指示書.md
  • docs/technical/Todo.md
  • docs/technical/*（設計資料）

目的

青空文庫HTMLを LuaLaTeX で PDF 化し、デバイス別の読みやすさを確保すること。Web UI から配色とデバイスを選択し、TeX/PDF 生成まで実行できます。

現在の構成方針

  • 起動エントリ: aozoratex.py, app.py
  • 設定: config/global_settings.*.ini / config/device_settings.*.ini
  • 実装本体: src/
  • 入力: data/
  • 出力: out/
  • Web静的資産: static/
  • LaTeXテンプレート群: latex_templates/
  • HTMLテンプレート群: html_templates/
  • ツール群: tools/
  • Markdownドキュメント: docs/

推奨運用

  • 通常利用は Web UI（app.py）か CLI（aozoratex.py）を使う。
  • 生成物の中間ログや一時ファイルは out/ 以下へ集約する。
  • 仕様メモは docs/ 配下に追記し、必要に応じて本HTMLに反映する。
