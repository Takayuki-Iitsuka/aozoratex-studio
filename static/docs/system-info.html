AozoraTeX Studio システム情報

確認日：2026年（皇紀2686年・令和8年）7月4日

このページでは、本システムで使用している言語・フレームワーク・ライブラリ・ツールの最新の概要とバージョン情報を詳細に記載しています。

システム概要

AozoraTeX Studio は、青空文庫のHTML/XHTMLを高品質な縦書きPDFへ変換するツールです。
アーキテクチャは「Next.js (App Router) + API Routes によるフロントエンド」と「Python による変換エンジン・ブリッジ」、そして「LuaLaTeX (jlreq)」の高度なハイブリッド構成になっています。

主な言語

  • TypeScript / JavaScript（UIとNext.js API基盤）
  • Python 3.11+（変換処理のコア、検証、ブリッジ）
  • LaTeX / Lua（組版記述、HarfBuzz連動）
  • HTML + Tailwind CSS v4（スタイリング）

主なフレームワーク

  • Next.js 16（App Router）
  • React 19
  • jlreq（LaTeX文書クラス、縦組）
