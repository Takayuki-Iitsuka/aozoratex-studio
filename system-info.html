AozoraTeX Studio システム情報

確認日：2026年（皇紀2686年・令和8年）7月3日

このページでは、本システムで使用している言語・フレームワーク・ライブラリ・ツールの概要とバージョン情報を詳細に記載しています。

システム概要

AozoraTeX Studio は、青空文庫のHTML/XHTMLを高品質な縦書きPDFへ変換するツールです。
アーキテクチャは「Next.js（フロントエンド + API） + Pythonブリッジ + LuaLaTeX（組版エンジン）」のハイブリッド構成になっています。

主な言語

  • Python 3（変換処理のコア）
  • TypeScript / JavaScript（UIとAPI）
  • LaTeX / Lua（組版記述）
  • HTML + CSS

主なフレームワーク

  • Next.js 15（App Router）
  • React 19
  • jlreq（LaTeX文書クラス、縦組）

補足・バージョン確認方法

  • Python依存： uv pip list で確認（uv 作成の .venv に pip は同梱されない。requirements.txt は最低バージョン指定）

  • TeX環境： lualatex --version および latexmk -v
