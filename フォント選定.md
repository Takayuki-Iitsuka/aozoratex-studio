### 思考プロセス (Thinking Process)

1.  **要件定義 (Requirements Definition)**:
    史学および古代中国文化を扱う文書では、JIS第1・第2水準に収まらない旧字体、異体字、拡張漢字（外字）が頻出します。言語学やテキスト解析において文字の出現頻度はZipfの法則 (Zipf's law) すなわち $f(k) \propto \frac{1}{k}$ に従うとされますが、頻度が極めて低い特殊な漢字であっても、学術文書においては正確にレンダリング (Rendering: 描画) されることが不可欠です。
2.  **評価基準 (Evaluation Criteria)**:
    総合的なフォントの評価値 $S$ を以下のように定義します。
    $$S = \alpha R + \beta C + \gamma A$$
    （$R$: Readability / 可読性、$C$: Compatibility / 外字・環境依存文字への対応力、$A$: Aesthetics / 文学的な美しさ。$\alpha, \beta, \gamma$ は重み付け）
3.  **絞り込み (Filtering)**:
    Windows標準フォントおよびTeXLive収録の日本語フォント群から、重み付けのバランスが異なる「本文用 (Body Text)」と「タイトル用 (Title / Heading)」を各3つ抽出します。

-----

### 本文用のフォント選定 (Body Text Fonts)

長文を読ませるための可読性 ($R$) と、史学科で要求される文字の網羅性 ($C$)、および文学的な美しさ ($A$) を重視した選定です。

| フォント名 (LaTeXコマンド例) | 開発元 / 特徴 |
| :--- | :--- |
| **IPAmjMincho**<br>`\setmainjfont{IPAmjMincho}` | **独立行政法人情報処理推進機構 (IPA)**<br>戸籍統一文字や住基ネット統一文字など約6万字を収録。異体字や古代中国関連の特殊文字表示において圧倒的な強さを誇ります。 |
| **Shippori Mincho**<br>`\setmainjfont{Shippori Mincho}` | **Fontdasu**<br>東京築地活版製造所の五号系活字をベースにしたオールドスタイル (Old Style: 古風な様式) の明朝体。文学作品において非常に評判が高いです。 |
| **Yu Mincho**<br>`\setmainjfont{Yu Mincho}` | **字游工房 (Microsoft Windows標準)**<br>「時代小説が組めるような明朝体」をコンセプトに開発された高品質フォント。 |

#### 各フォントのメリット・デメリット・リスク

  * **IPAmjMincho**
      * **メリット (Merits)**: 圧倒的な外字・異体字対応力 ($C$ が最大)。史学の論文において、文字化けや表示不能（豆腐文字）を防ぐ最適解です。
      * **デメリット (Demerits)**: 行政文書向けに開発されたため、文学的な「潤い」や「美しさ」($A$) は控えめです。
      * **リスク (Risks)**: ウェイト (Weight: 太さ) のバリエーションがないため、本文中での強調表現（太字化）が困難です。
  * **Shippori Mincho**
      * **メリット (Merits)**: 活版印刷のインクの溜まりを再現したような曲線美があり、文学的なエッセイや歴史的テーマの文書に極めて高い没入感を与えます ($A$ が最大)。
      * **デメリット (Demerits)**: JIS第3水準以降の特殊な拡張漢字・中国語圏の漢字への対応力 ($C$) は大企業製フォントに劣ります。
      * **リスク (Risks)**: 収録されていない古代文字を使用した場合、その文字だけ別のフォントに置き換わるフォールバック (Fallback: 代替処理) が発生し、デザインの統一性が崩れる危険性があります。
  * **Yu Mincho**
      * **メリット (Merits)**: 可読性 ($R$) と美しさ ($A$) のバランスが良く、Windows環境に標準搭載されているため環境構築が容易です。
      * **デメリット (Demerits)**: 広く普及しているため、視覚的な「特別感」や「個性」は薄くなります。
      * **リスク (Risks)**: 異体字セレクタ (IVS: Ideographic Variation Sequence) への対応は充実していますが、IPAmjMinchoほどの極端な網羅性はありません。

### タイトル用のフォント選定 (Title Fonts)

視認性 (Visibility) と、縮小・拡大時にも破綻しない構造の強さ、そして読者の目を引くインパクトを重視した選定です。

| フォント名 (LaTeXコマンド例) | 開発元 / 特徴 |
| :--- | :--- |
| **Source Han Serif Heavy**<br>`\setmainjfont{Source Han Serif}[Weight=Heavy]` | **Adobe / Google**<br>Pan-CJK (汎CJK: 日中韓統合) フォントである源ノ明朝の極太ウェイト。 |
| **Yu Mincho Demibold**<br>`\setmainjfont{Yu Mincho}[Weight=Demibold]` | **字游工房 (Microsoft Windows標準)**<br>游明朝の太字ウェイト。 |
| **BIZ UDPMincho**<br>`\setmainjfont{BIZ UDPMincho}` | **モリサワ**<br>ユニバーサルデザイン (Universal Design) の思想を取り入れた、視認性に極めて優れる明朝体。 |

#### 各フォントのメリット・デメリット・リスク

  * **Source Han Serif Heavy**
      * **メリット (Merits)**: 日中韓の漢字を網羅しており、タイトルに中国語の特殊文字が混ざっても力強く美しい極太の明朝体で描画可能です。
      * **デメリット (Demerits)**: 非常に力強い反面、繊細で静かな文学作品の表紙としては主張が強すぎる場合があります。
      * **リスク (Risks)**: LuaLaTeXでの読み込み時に正しくウェイト指定 (Weight specification) を行わないと、標準の太さで出力されてしまう設定ミスが起こりやすいです。
  * **Yu Mincho Demibold**
      * **メリット (Merits)**: 本文を「Yu Mincho」にした場合、完全な統一感 (Consistency) を保ったまま上品なタイトルを作成できます。
      * **デメリット (Demerits)**: ポスターやプレゼン資料のような、遠目からでも目立つ強烈な太さはありません。
      * **リスク (Risks)**: 他の個性的なフォントと組み合わせた場合、タイトルが本文の印象に負けてしまう可能性があります。
  * **BIZ UDPMincho**
      * **メリット (Merits)**: 誰にとっても読みやすく、濁点や半濁点、複雑な画数の漢字も潰れずにクリアに印字されます。
      * **デメリット (Demerits)**: 近代的で実用的なデザインのため、古代中国文化や歴史を扱う上での「古風な雰囲気 (Antique feel)」は損なわれます。
      * **リスク (Risks)**: プロポーショナルフォント (Proportional font: 文字ごとに幅が異なるフォント) であるため、タイトルの文字間隔 (Kerning: カーニング) が自動で詰まり、意図したレイアウトにならない場合があります。

### 別の視点・潜在的なリスク (Counter-arguments & Potential Risks)

「美しさ」と「学術的な正確性（外字対応）」を一つのフォントで完全に両立させることは、現代のタイポグラフィ (Typography: 文字配列の技術) において**極めて困難**であるという反論が成り立ちます。

史学科の学生としての専門的なレポート作成において、「見た目が綺麗で評判の良いフォント（例：しっぽり明朝）」をメインに据えた場合、必然的に古代中国の特殊な漢字や異体字が抜け落ちるリスクを抱えます。
これを回避するための現実的な運用策 (Workaround) として、\*\*「基本の本文は文学的なフォント (Shippori Minchoなど) に設定し、該当する特殊な外字が登場した箇所だけを局所的に学術用フォント (IPAmjMincho) で囲んで出力する」\*\*というハイブリッドなマークアップ (Markup: タグ付け) をLuaLaTeX上で行う運用を強く推奨します。
