# 指示書

このプロジェクトに
CSS Framework (CSSフレームワーク)
Bootstrap
<https://getbootstrap.jp/>
の最新版を導入します。
理由は、見栄えを良くするためです。

Bootstrapの最新版を導入するので、
その前に、static　フォルダ内にあるすべてのについて整理整頓、適切なフォルダ分け、ファイル名を英語し内容にあった名称変更、不要なファイルの削除、内容の精査を行ってください。
特に、static\docs　フォルダにあるファイルについては、ファイルの内容を確認し、内容にあったファイル名を英語に変更し、統合や分割、削除を行ってください。static\docs\markdown　フォルダがありますが、内容別フォルダに整理整頓を行ってください。
